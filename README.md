# BCG GenAI Financial Chatbot

This project was developed as part of the **BCG Generative AI Virtual Experience Program on Forage**.

## Project Overview

The objective of this project is to build an **AI-powered financial chatbot** capable of analyzing company financial data and providing simplified insights.

The chatbot processes financial information from **10-K and 10-Q reports** and applies **rule-based logic** to generate user-friendly financial responses.

## Technologies Used

- Python
- Pandas
- Financial Data Analysis
- Rule-based Chatbot Logic

## Features

- Extracts insights from company financial reports
- Processes structured financial datasets
- Converts complex financial information into simplified explanations
- Demonstrates applications of AI in financial consulting

## Learning Outcomes

Through this simulation I gained experience in:

- Python programming
- Data manipulation using Pandas
- Interpreting financial reports
- Designing rule-based chatbot systems
